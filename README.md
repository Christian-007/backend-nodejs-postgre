# backend-nodejs-postgre
A Simple Todos BE Server created in JavaScript using NodeJS.

## Technology used:
- NodeJS
- PostgreSQL (hosted in ElephantSQL)
- `express`
- `sequelize`
- `nodemon`
- `morgan`
- `prettier`
